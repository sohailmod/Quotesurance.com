## Design System

### Color Palette:
- **Primary**: Blue (#3B82F6) to Purple (#9333EA) gradients
- **Accents**: Pink (#EC4899), Cyan (#06B6D4)
- **Background**: Slate 900 (#0F172A) with purple tints
- **Text**: White primary, Slate 300 secondary

### Typography:
- Headlines: Bold, tight letter-spacing
- Body: Slate 300, relaxed line-height
- Buttons: Semibold, 16-18px

### Components:
- **Cards**: White/10 background, white/10 border, rounded-2xl
- **Buttons**:
  - Primary: Blue-purple gradient, rounded-xl
  - Secondary: White/10 bg, white/20 border
- **Icons**: Lucide React icons, sized 24-32px for features

### Layout:
- Max width: 1280px (7xl) for main content
- Responsive breakpoints: sm, md, lg
- Consistent spacing: py-20 sections, px-6 containers
